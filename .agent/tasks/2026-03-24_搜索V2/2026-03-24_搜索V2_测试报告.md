# 搜索 V2 测试报告

> 日期：`2026-03-25`
> 任务：搜索 V2

## 测试范围

覆盖搜索 V2 的 P5 测试验收，包含四类：
- 单元测试：索引构建口径、搜索数据映射、URL 同步关键实现核对
- 冒烟测试：`/search` 页面首屏可加载、索引可返回、基础搜索路径可用
- 功能测试：`/search?q=` 首屏回填、清空移除 `q`、posts + projects 混合结果、浏览器前进/后退恢复
- 回归测试：`npm run lint`、`npm run build`、`public/search-index.json` 构建产物结构与范围

## 测试结果

### 单元测试

- [✔] 索引仅包含有效公开内容：核对 `scripts/build-search-index.ts` 复用 `getPosts()` / `getProjects()`，并检查 `public/search-index.json`，确认 `bad-invalid-example` 不在索引内
- [✔] 搜索条目统一模型正确：核对 `src/lib/search.ts`，确认输出 `type / href / section / title / excerpt / tags / date / searchText`，且 posts / projects 分别映射到 `/blog/[slug]` 与 `/projects/[slug]`
- [✔] URL 双向同步关键逻辑存在：核对 `src/components/search-box.tsx`，确认存在 `urlQuery -> query` 回填 effect、300ms debounce、清空时删除 `q` 参数、显式 `history.pushState` 写入历史栈

### 冒烟测试

- [✔] `/search` 页面可正常返回：`curl -I http://127.0.0.1:3000/search` 可达；真实浏览器快照能进入搜索页首屏
- [✔] 搜索索引可正常返回：`curl -i http://127.0.0.1:3000/search-index.json` 返回 `200 OK`
- [✔] 搜索页首屏基础链路可用：真实浏览器（Playwright）访问 `http://localhost:3000/search?q=agent` 后，输入框回填为 `agent`，结果区展示 `找到 2 条相关内容`

### 功能测试

- [✔] `/search?q=agent` 首屏回填：真实浏览器验证输入框值为 `agent`，并自动展示结果
- [✔] posts + projects 混合结果正确：真实浏览器搜索 `agent` 时同时出现文章 `/blog/agent-gongzuoliu-yu-bianjie` 与项目 `/projects/openclaw-workflow`
- [✔] 清空移除 `q`：真实浏览器将搜索框清空后，URL 恢复为 `http://localhost:3000/search`，页面展示空查询提示“输入关键词开始搜索文章和项目”
- [✔] 浏览器前进/后退时输入框 / URL / 结果一致：真实浏览器执行 `agent -> node -> 后退 -> 前进`，后退后恢复到 `?q=agent` 且结果恢复为文章+项目混合；前进后恢复到 `?q=node` 且结果恢复为两个项目 `/projects/openclaw-workflow`、`/projects/legacy-tools`
- [✔] 搜索结果跳转口径正确：代码核对与浏览器断言均确认结果链接使用 `href` 字段，而非写死博客路径
