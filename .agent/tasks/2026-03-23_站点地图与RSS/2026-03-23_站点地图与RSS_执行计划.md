# 站点地图与 RSS 执行计划

> 日期：`2026-03-23`
> 任务：站点地图与 RSS
> 状态：进行中

## 目标

为博客添加 sitemap.xml 和 rss.xml 两个公开路由，支持 SEO 爬取和 RSS 阅读器订阅。

## 上下文

- 功能：SEO / RSS
- 相关文件：
  - `src/lib/site.ts` — siteUrl 等配置
  - `src/lib/posts.ts` — getPosts() 读取已发布文章
  - `content/posts/*.mdx` — 源文件

## 验收标准

- [ ] `/sitemap.xml` 返回 200，包含所有已发布文章条目
- [ ] `/rss.xml` 返回 200，包含所有已发布文章条目
- [ ] URL 使用 `siteConfig.siteUrl` 作为基准
- [ ] 文章链接格式为 `/blog/[slug]`
- [ ] `npm run lint` 通过
- [ ] `npm run build` 通过

## 决策日志

- Decision: 使用 Next.js App Router 的 `route.ts` 生成动态 sitemap 和 RSS
  原因：无需额外依赖，SSR/ISR 可控，与现有 posts.ts 无缝集成
  日期/作者：2026-03-23 / builder-codex

- Decision: RSS 使用 Atom 格式（兼容 RSS 2.0 读取器）
  原因：广泛兼容，最简实现

## 实施步骤

### Step 1: 创建 sitemap.xml route handler

- 文件：`src/app/sitemap.xml/route.ts`
- 内容：
  - import getPosts() + siteConfig
  - 生成 XML，包含 homepage、blog list、每篇文章
  - 返回 text/xml

### Step 2: 创建 rss.xml route handler

- 文件：`src/app/rss.xml/route.ts`
- 内容：
  - import getPosts() + siteConfig
  - 生成 RSS 2.0 XML
  - 包含 title、link、description、pubDate、guid
- 返回 text/xml
