```css
  --color-dark-bg-surface: #111827;
  --color-dark-bg-subtle: #1e293b;
  --color-dark-text-primary: #f1f5f9;
  --color-dark-text-secondary: #94a3b8;
  --color-dark-text-muted: #64748b;
  --color-dark-accent: #60a5fa;
  --color-dark-accent-hover: #93c5fd;
  --color-dark-accent-subtle: #1e3a5f;
  --color-dark-border: #1e293b;
}
```

**同时更新暗色模式 dark class 覆盖：**
在 `globals.css` 底部添加：

```css
.dark {
  --color-bg-primary: var(--color-dark-bg-primary);
  --color-bg-surface: var(--color-dark-bg-surface);
  --color-bg-subtle: var(--color-dark-bg-subtle);
  --color-text-primary: var(--color-dark-text-primary);

  --color-accent-hover: var(--color-dark-accent-hover);
  --color-accent-subtle: var(--color-dark-accent-subtle);
  --color-border: var(--color-dark-border);
}
```

- **预期结果：** 任意页面 body 加 `.dark` class 后整体色调切换
