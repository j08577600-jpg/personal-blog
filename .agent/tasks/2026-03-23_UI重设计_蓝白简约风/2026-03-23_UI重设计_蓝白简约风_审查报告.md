## 发现的问题

### 🚨 阻塞问题：1 个（已修复）

**1. 文章页标签胶囊不可点击**
- **位置：** `src/components/article-renderer.tsx` 第 27–33 行
- **问题：** 标签使用 `<span>` 渲染，导致用户无法从文章详情页点击标签进入标签聚合页。这违反了「标签可点击」的交互预期。
- **修复：** 改为 `<Link href={/blog/tags/${encodeURIComponent(tag)}}>`，与 PostCard 和标签详情页保持一致。
- **Commit：** `c09a121` ✅ 已推送

```diff
- <span
-   key={tag}
-   className="inline-flex items-center rounded-full ..."
- >
-   {tag}
- </span>
+ <Link
+   key={tag}
+   href={`/blog/tags/${encodeURIComponent(tag)}`}
+   className="inline-flex items-center rounded-full ..."
+ >
+   {tag}
+ </Link>
```
