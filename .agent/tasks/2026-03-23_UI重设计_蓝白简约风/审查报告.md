**3. Dashboard 页面宽度超出规范**
- **位置：** `src/app/dashboard/page.tsx`
- **现状：** Dashboard 使用 `max-w-5xl`，而公开页面群（首页、博客、标签）统一使用 `max-w-4xl`。
- **说明：** Dashboard 是内部工具页，宽度略宽有合理性。方案设计.md 中未明确覆盖 Dashboard 宽度，属于模糊地带。
- **建议：** 保持现状。如需统一，可改为 `max-w-4xl`，不影响功能。

**4. Dashboard 保留语义状态色**
- **位置：** `src/app/dashboard/page.tsx`
- **现状：** Status badge 和 StatCard 使用 `bg-green-50 text-green-700` 等 Tailwind 语义色（非 CSS 变量）。
- **说明：** 这是合理的语义选择（绿色=发布、琥珀=草稿、红色=错误），与蓝白系统不冲突。Dashboard 非公开页面，无暗色变体一致性问题。
- **建议：** 保持现状，无需修改。
