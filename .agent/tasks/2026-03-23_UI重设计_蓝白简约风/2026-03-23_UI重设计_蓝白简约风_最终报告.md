## 三、技术实现摘要

### 3.1 核心设计系统

**CSS 变量 + Tailwind v4 `@theme` 方案**，10 个语义化 token，亮/暗双态完整覆盖。

```
亮色 accent：#2563eb    暗色 accent：#60a5fa
暗色主背景：#0c1222     暗色 surface：#111827
```

**暗色模式机制：** `.dark` class on `<html>` + `ThemeScript`（`<head>` 最早注入，防闪烁）+ `useSyncExternalStore`（ThemeToggle，非 useEffect setState）。
