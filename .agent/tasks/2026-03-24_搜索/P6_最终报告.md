## 决策记录

| # | 决策 | 理由 | 日期/作者 |
|---|------|------|-----------|
| D1 | 采用 Fuse.js 客户端搜索而非服务端 API | V1 文件驱动原则，无数据库依赖，部署简单，零冷启动 | 2026-03-24 / planner-codex |

| D3 | V1 仅索引 title + excerpt + tags，不含全文 | 保持 JSON 体积可控；全文索引可 V2 扩展 | 2026-03-24 / planner-codex |
| D4 | prebuild + build 脚本均触发索引生成 | prebuild 在 `next build` 之前触发；build 内再次调用确保覆盖率 | 2026-03-24 / builder-codex |
| D5 | Header 搜索入口仅使用图标（sr-only 文字）| Header 空间有限，图标搜索入口是常见模式 | 2026-03-24 / builder-codex |
