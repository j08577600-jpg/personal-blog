## 非阻塞性问题（Non-Blocking）

### NP-1：OpenGraph tags 与 UI displayTags 不一致

**位置**：`src/app/projects/[slug]/page.tsx` → `generateMetadata`

**描述**：`generateMetadata` 传入 `tags: project.tags`，而页面 UI 显示的是 `displayTags = project.tech ?? project.tags`。若某项目同时有两字段且内容不同，爬虫看到的标签与页面显示不一致。

**建议**：对齐 metadata：
```ts
tags: displayTags,  // 与 UI 保持一致
```

**严重性**：低（内容相关性，无功能影响）

---

### NP-2：ArticleRenderer 与详情页 prose 样式重复

**位置**：`src/components/article-renderer.tsx` 与 `src/app/projects/[slug]/page.tsx`

**描述**：两处 `className="prose prose-lg dark:prose-invert..."` 样式块完全相同（约 15 行），`ExternalLinkIcon`/`GitHubIcon`/`LiveIcon` 组件也在两处重复定义。

**建议**：V2 将 prose 样式抽取为共享 CSS 变量或专用组件；当前 V1 可接受，不影响功能。

**严重性**：低（DRY 原则，但无维护负担）

---

### NP-3：projects.ts 游离 import

**位置**：`src/lib/projects.ts` 末尾

**描述**：`import Link from "next/link"` 未被使用（Link 在 `project-card.tsx` 中使用，不在此文件）。

**建议**：删除该行。

**严重性**：极低（lint 会自动捕获，但手动清理更干净）
