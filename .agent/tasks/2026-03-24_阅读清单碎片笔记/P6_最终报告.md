## 三、架构决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 内容存储 | 文件驱动（MDX）| 复用 posts/projects 现有架构，无数据库依赖 |
| 工具函数 | 独立 lib（不泛型复用 posts.ts）| reading-list 和 notes 字段差异大，V1 简单优先 |
| UI 组件 | 独立组件（不复用 PostCard/ProjectCard）| 字段差异大，独立组件更清晰，ESLint 无报错 |
| RSS feed | 不含 reading-list/notes | RSS 以博客文章为主，辅助内容不入 feed |
| sitemap | 接入（priority 0.7 / weekly）| SEO 可发现性 |
