## 主审结论（reviewer-minimax）

### 必改项

_无。_

### 可改项

1. **文章页 OpenGraph `publishedTime` 格式**：当前 `post.date` 为 `yyyy-MM-dd` 字符串，传入 `publishedTime` 时直接作为字符串使用。OpenGraph 规范建议 ISO 8601（如 `2026-03-23T00:00:00Z`），部分爬虫可能对简化格式兼容不佳。**低优先级**：当前格式对主流平台兼容性尚可，V2 可考虑转换。

2. **文章页 metadata 缺少 `authors[].url`**：`authors` 数组目前仅设 `name`，未提供 GitHub 个人主页 URL。**低优先级**：SEO 影响轻微，可作为 V2 优化项。

3. **`siteConfig.siteUrl` 硬编码**：当前值 `https://blog.chenjilan.com` 写在 `src/lib/site.ts` 中，域名变更需手动修改。V2 可考虑通过环境变量注入。

### 是否允许进入测试
